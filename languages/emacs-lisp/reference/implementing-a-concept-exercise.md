# How to implement an Emacs Lisp concept exercise

To implement an Emacs Lisp concept exercise, the following files must be added:

```
languages
└── emacs-lisp
    └── exercises
        └── concept
            └── <SLUG>
                ├── .docs
                |   ├── instructions.md
                |   ├── introduction.md
                |   ├── hints.md
                |   └── after.md
                ├── .meta
                |   |── design.md
                |   └── Example.el
                ├── <NAME>.el
                └── <NAME>_test.el
```
